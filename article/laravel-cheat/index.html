Laravel Cheat Sheet

Kumpulan command yang sering dipakai — dari setup project hingga Spatie package dan deployment. Klik kode untuk copy.

🚀
New Project
Buat project baru via Composer
composer create-project laravel/laravel nama-project
Buat project via Laravel Installer
laravel new nama-project
Versi spesifik
composer create-project laravel/laravel nama-project "11.*"
Jalankan dev server
php artisan serve
php artisan serve --port=8080
Generate app key
php artisan key:generate
⚙️

List semua command
php artisan list
php artisan help make:model
Clear cache semua
php artisan optimize:clear
# atau satu per satu:
php artisan cache:clear
php artisan config:clear
php artisan route:clear
php artisan view:clear

Cache untuk production
php artisan optimize

Migration & Seeder
Buat migration

php artisan migrate

Buat & jalankan seeder

php artisan db:seed
php artisan db:seed --class=UserSeeder
Buat factory

Spatie — Permission & Role spatie/laravel-permission
📦 Package untuk manajemen role dan permission berbasis database. Mendukung multiple guard.
Install
composer require spatie/laravel-permission
php artisan vendor:publish --provider="Spatie\Permission\PermissionServiceProvider"
php artisan migrate
Tambahkan trait ke model User
use Spatie\Permission\Traits\HasRoles;

class User extends Authenticatable
{
    use HasRoles;
}
Buat role & permission via seeder / tinker
// Buat permission
Permission::create(['name' => 'edit articles']);

// Buat role
Role::create(['name' => 'admin']);

// Assign permission ke role
$role->givePermissionTo('edit articles');
$role->syncPermissions(['edit articles', 'delete articles']);

// Assign role ke user
$user->assignRole('admin');
$user->assignRole(['admin', 'writer']);
$user->syncRoles(['admin']);

// Cek permission
$user->hasRole('admin');
$user->hasPermissionTo('edit articles');
$user->can('edit articles');

// Revoke
$user->removeRole('admin');
$user->revokePermissionTo('edit articles');
Middleware di route
Route::middleware(['role:admin'])->group(function() { ... });
Route::middleware(['permission:edit articles'])->group(...);
Route::middleware(['role_or_permission:admin|edit articles'])->group(...);

// Di blade
@role('admin') ... @endrole
@can('edit articles') ... @endcan
Reset cache permission
php artisan permission:cache-reset

Spatie — Media Library spatie/laravel-medialibrary
📦 Upload dan kelola file/gambar yang terasosiasi dengan model. Support konversi gambar otomatis.
Install
composer require spatie/laravel-medialibrary
php artisan vendor:publish --provider="Spatie\MediaLibrary\MediaLibraryServiceProvider" --tag="medialibrary-migrations"
php artisan migrate
php artisan storage:link
Tambahkan interface ke model
use Spatie\MediaLibrary\HasMedia;
use Spatie\MediaLibrary\InteractsWithMedia;

class Product implements HasMedia
{
    use InteractsWithMedia;

    public function registerMediaCollections(): void
    {
        $this->addMediaCollection('images')
             ->singleFile();          // opsional: hanya 1 file

        $this->addMediaCollection('documents');
    }

    // Konversi otomatis (butuh intervention/image)
    public function registerMediaConversions(Media $media = null): void
    {
        $this->addMediaConversion('thumb')
             ->width(300)->height(300);
    }
}
Upload file
// Upload dari request
$product->addMediaFromRequest('photo')
         ->toMediaCollection('images');

// Upload dari URL
$product->addMediaFromUrl('https://example.com/image.jpg')
         ->toMediaCollection('images');

// Upload dari path lokal
$product->addMedia(storage_path('file.pdf'))
         ->toMediaCollection('documents');

// Simpan dengan nama custom
$product->addMediaFromRequest('photo')
         ->usingFileName('custom-name.jpg')
         ->toMediaCollection('images');
Ambil & hapus media
// Ambil URL
$product->getFirstMediaUrl('images');
$product->getFirstMediaUrl('images', 'thumb'); // konversi

// Ambil semua
$product->getMedia('images');

// Hapus semua di collection
$product->clearMediaCollection('images');

Spatie — Activity Log spatie/laravel-activitylog
Install
composer require spatie/laravel-activitylog
php artisan vendor:publish --provider="Spatie\Activitylog\ActivitylogServiceProvider" --tag="activitylog-migrations"
php artisan migrate
Tambahkan trait ke model
use Spatie\Activitylog\Traits\LogsActivity;
use Spatie\Activitylog\LogOptions;

class Product extends Model
{
    use LogsActivity;

    public function getActivitylogOptions(): LogOptions
    {
        return LogOptions::defaults()
            ->logOnly(['name', 'price'])
            ->logOnlyDirty()
            ->dontSubmitEmptyLogs();
    }
}
Log manual
activity()->log('User melakukan sesuatu');

activity()
    ->performedOn($product)
    ->causedBy($user)
    ->withProperties(['key' => 'value'])
    ->log('Product diedit');

// Ambil log
Activity::all();
$product->activities();

Laravel Passport (OAuth2) laravel/passport
⚠️ Gunakan Passport untuk OAuth2 penuh (client credentials, authorization code). Untuk API token sederhana, pertimbangkan Sanctum.
Install
composer require laravel/passport
php artisan migrate
php artisan passport:install           # generate encryption keys + clients
php artisan passport:install --uuids  # gunakan UUID
Tambahkan trait ke model User
use Laravel\Passport\HasApiTokens;

class User extends Authenticatable
{
    use HasApiTokens;
}
Config auth.php — guard api
// config/auth.php
'guards' => [
    'api' => [
        'driver'   => 'passport',
        'provider' => 'users',
    ],
],
Manage keys & clients
php artisan passport:keys              # regenerate keys
php artisan passport:client            # buat client baru (interaktif)
php artisan passport:client --personal # personal access client
php artisan passport:client --password # password grant client
php artisan passport:client --client   # client credentials
Middleware & scope di route
Route::middleware('auth:api')->get('/user', ...);
Route::middleware(['auth:api', 'scopes:read-orders'])->get(...);
Route::middleware(['auth:api', 'scope:read-orders,write-orders'])->get(...);

Laravel Sanctum (API Token) laravel/sanctum
Install
composer require laravel/sanctum
php artisan vendor:publish --provider="Laravel\Sanctum\SanctumServiceProvider"
php artisan migrate
Tambahkan trait ke model User
use Laravel\Sanctum\HasApiTokens;

class User extends Authenticatable
{
    use HasApiTokens;
}
Buat & hapus token
// Buat token
$token = $user->createToken('token-name')->plainTextToken;
$token = $user->createToken('token-name', ['read', 'write'])->plainTextToken;

// Hapus token saat ini
$request->user()->currentAccessToken()->delete();

// Hapus semua token
$user->tokens()->delete();

Queue & Job
Setup queue database
php artisan queue:table
php artisan migrate
Buat job
php artisan make:job SendEmailJob
php artisan make:job SendEmailJob --sync  # synchronous
php artisan make:mail OrderShipped --markdown=emails.orders.shipped
Jalankan worker
php artisan queue:work
php artisan queue:work --tries=3
php artisan queue:work --queue=high,default
php artisan queue:work --timeout=60
php artisan queue:listen
php artisan queue:restart    # restart worker setelah deploy
php artisan queue:failed     # lihat failed jobs
php artisan queue:retry all  # retry semua failed
php artisan queue:flush      # hapus semua failed

Task Scheduling
Buat command terjadwal
php artisan make:command SendDailyReport

* * * * * cd /path-to-project && php artisan schedule:run >> /dev/null 2>&1

Cache
Cache via code
Cache::put('key', 'value', now()->addMinutes(10));
Cache::get('key', 'default');
Cache::has('key');
Cache::forget('key');
Cache::flush();

// Remember — auto cache atau ambil dari closure
$value = Cache::remember('users', 3600, fn() => User::all());
$value = Cache::rememberForever('key', fn() => ...);

// Tags (butuh Redis/Memcached)
Cache::tags(['products'])->put('key', $val, 600);
Cache::tags(['products'])->flush();

Deploy Checklist
✅ Urutan command yang aman saat deploy ke production.
Deploy sequence
git pull origin main

composer install --optimize-autoloader --no-dev

php artisan down                   # maintenance mode

php artisan migrate --force
php artisan db:seed --force       # jika ada seeder baru

php artisan optimize:clear
php artisan optimize
php artisan storage:link

php artisan queue:restart          # restart workers
php artisan permission:cache-reset # jika pakai spatie/permission

php artisan up                     # matikan maintenance mode
Maintenance mode
php artisan down
php artisan down --secret="my-secret"  # akses via /my-secret
php artisan down --render="errors.503"
php artisan up
