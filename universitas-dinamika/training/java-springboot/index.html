// Kurikulum

Yang Akan Kamu Pelajari

Kurikulum terstruktur 45 hari mengikuti Spring Boot developer roadmap—dari fondasi Java modern hingga deployment aplikasi production.

☕

// Week 1 · Hari 1–7

Core Java & OOP Foundation

Java 17/21 OOP Principles Collections & Generics Lambda & Streams Exception Handling Maven / Gradle

Memastikan fondasi Java yang kuat: syntax modern, paradigma OOP, serta tooling yang dipakai di dunia profesional.

🌱

// Week 1–2 · Hari 5–14

Spring Core & Dependency Injection

IoC Container Beans & Scopes @Component, @Service @Autowired / DI ApplicationContext Spring AOP

Memahami cara kerja Spring Framework dari dalam: IoC, DI, dan bagaimana Spring mengelola lifecycle komponen aplikasi.

🚀

// Week 2–3 · Hari 8–21

Spring Boot Basics & REST API

Spring Boot Setup Auto-configuration @RestController @RequestMapping Request/Response Body Validation (Bean) Exception Handling Swagger / OpenAPI

Membangun RESTful API yang bersih dan terdokumentasi dengan baik—inti dari pengembangan backend modern.

🗄️

// Week 3–4 · Hari 15–28

Database, JPA & Hibernate

Spring Data JPA Hibernate ORM Entity & Relationships JPQL & Criteria API Repository Pattern Database Migration MySQL / PostgreSQL Transaction Management

Mengelola data dengan benar: relasi antar tabel, query yang efisien, dan strategi transaksi untuk integritas data.

🔐

// Week 4–5 · Hari 22–35

Security & JWT Authentication

Spring Security JWT (JSON Web Token) Authentication Flow Authorization & Roles Password Encoding Filter Chain CORS Configuration

Mengamankan API dengan Spring Security dan JWT—standar industri untuk autentikasi stateless pada aplikasi backend modern.

🏗️

// Week 5–6 · Hari 29–42

Clean Architecture & Best Practices

Layered Architecture DTO Pattern Service Layer Repository Pattern Unit Testing (JUnit) Integration Testing Logging (SLF4J) Error Handling Strategy

Menulis kode yang maintainable, testable, dan scalable—pola arsitektur yang digunakan tim profesional di perusahaan teknologi.

🚢

// Week 6 · Hari 36–45

Deployment & Final Project

Docker & Containerization Docker Compose Environment Config CI/CD Basics VPS / Cloud Deploy Final Project Code Review

Mengemas dan mendeploy aplikasi ke server production menggunakan Docker—lalu membangun final project lengkap sebagai portofolio.

// Prasyarat

Siapa yang Cocok Ikut?

Bootcamp ini dirancang untuk peserta yang sudah memiliki dasar dan siap naik level ke pengembangan backend profesional.

💻

Basic Programming

Sudah memahami konsep dasar pemrograman. Pengalaman dengan Java sangat dianjurkan, namun developer dengan bahasa lain tetap dapat mengikuti dengan penyesuaian singkat.

🧩

Memahami OOP

Mengerti fundamental Object-Oriented Programming: class, object, inheritance, encapsulation, dan polymorphism. Konsep ini menjadi fondasi seluruh materi Spring.

🔥

Siap Hands-on

Komitmen untuk belajar aktif selama 45 hari. Setiap sesi mengharuskan coding langsung, mengerjakan latihan, dan berkontribusi pada proyek nyata.
